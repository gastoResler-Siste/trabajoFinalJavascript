Trabajo Practico 3 - Resler Gaston

Primera Parte


1. Crea un arreglo de objetos que representen estudiantes con las propiedades "nombre", "edad" y "carrera". Luego, crea una función que tome este arreglo como argumento y devuelva el promedio de edad de los estudiantes.



2. Crea un objeto que represente una cuenta bancaria con las propiedades "nombre", "saldo" y "tipo de cuenta". Luego, crea una función que tome este objeto como argumento y devuelva el texto "Tu saldo es de [saldo] en tu cuenta [tipo de cuenta]".


3. Crea una función que tome un arreglo de objetos que representen libros con las propiedades "título", "autor" y "año de publicación". Luego, ordena el arreglo de libros por año de publicación de menor a mayor



4. Crea una función que tome un arreglo de objetos que representen productos con las propiedades "nombre", "precio" y "descuento". Luego, calcula el precio final de cada producto después de aplicar el descuento y devuelve un nuevo arreglo con los precios finales..

Segunda Parte

1. Crear un Formulario H T M L que incluya los siguientes campos: Nombre, correo electronico y telefono. Añadir un boton 'Validar'. Al hacer clic en el boton, el programa debe verificar que los tres campos se han completado, el telofono tienne menos de 11 digitos y el correo electronico contiene la '@'. si todas las condiciones se cumplen, mostrar un mensaje que diga:. 'La informacion es Correcta', de lo contrario mostrar 'Por favor intente Nuevamente'.








2. Crear un área de texto HTML y solicitar al usuario que ingrese un texto. Al presionar la tecla Enter, el programa deberá mostrar la cantidad de letras que tiene el texto ingresado en la pantalla.




3. Crear un documento HTML que contenga un DIV con el fragmento de un texto cualquiera. Crear un botón que si el texto esta visible que diga “Ocultar” y de lo contrario que diga “Mostrar”. Al hacer click tiene que hacer dicha acción.
